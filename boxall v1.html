Besoin d'aide ?

Notre équipe Boxall est disponible pour vous accompagner .

À propos de Boxall Boxall est une application de communication qui permet aux utilisateurs de rester connectés simplement, rapidement et efficacement. Notre objectif est de rapprocher les personnes en facilitant les appels entre différents réseaux grâce à la puissance d'Internet. Boxall : connecter les frontières, rapprocher les voix.
## Les Agents Boxall Les Agents Boxall sont des partenaires de proximité qui accompagnent les utilisateurs dans l'utilisation de nos services. Ils permettent notamment de : * aider à la création de comptes Boxall ; * accompagner les utilisateurs lors des recharges ; * expliquer le fonctionnement de l'application ; * apporter une assistance aux personnes ayant des difficultés avec les services numériques. Grâce aux Agents Boxall, chaque utilisateur peut bénéficier d'un accompagnement simple et accessible. ## Tarification Boxall Boxall propose des services conçus pour rester accessibles à tous. Les tarifs peuvent varier selon : * le type d'abonnement choisi ; * la durée des appels ; * les offres promotionnelles disponibles. Nos objectifs sont de proposer une communication de qualité à un prix adapté aux besoins de nos utilisateurs. ## Devenir Agent Boxall Toute personne souhaitant représenter Boxall peut devenir agent après validation de son profil. Les Agents Boxall bénéficient : * d'une visibilité dans leur zone ; * d'outils pour accompagner les utilisateurs ; * d'opportunités liées au développement du réseau Boxall.
